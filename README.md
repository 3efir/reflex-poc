# reflex-poc

Simple customization of [reflex](http://reflexgrid.com/docs/)

### Installing

```
git clone https://github.com/3efir/reflex-poc.git
cd reflex-poc
npm install
```

### Start & Watch

```
gulp serve
gulp watch
```

### Build

```
gulp styles (for dev with sourcemaps)
gulp styles --env prod (for production)
```
